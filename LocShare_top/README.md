# LocShare

Static landing page for the LocShare: GPS Locator app.

## Files

- `index.html` - homepage
- `styles.css` - page styling
- `assets/` - image assets
- `app-ads.txt` - app ads declaration

## App Ads

The app ads file should be served from the site root:

```text
/app-ads.txt
```

Expected content:

```text
google.com, pub-3615322193850391, DIRECT, f08c47fec0942fa0
```
